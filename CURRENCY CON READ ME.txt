 MVC Architecture

This application follows the **Model-View-Controller** design pattern:

  Model (`CurrencyModel`)
- Fetches exchange rates from API
- Manages currency data
- Performs conversion calculations
- Handles all business logic

 View (`CurrencyView`)
- Creates and manages UI components
- Displays data to users
- Captures user inputs
- Updates visual elements

Controller (`CurrencyController`)
- Connects Model and View
- Handles user actions
- Coordinates data flow
- Manages application state

 API Reference

This application uses the **ExchangeRate-API** for currency data:

```
Base URL: https://api.exchangerate-api.com/v4/latest/
Endpoint: /USD (or any base currency)
Response: JSON with exchange rates
```

**Example Response:**
```json
{
  "base": "USD",
  "rates": {
    "EUR": 0.9245,
    "GBP": 0.7892,
    "JPY": 149.52,
    "GHS": 15.45
  },
  "date": "2025-10-30"
}
```

Customization

Change default currencies:
```python
# In CurrencyView.populate_currencies()
self.from_combo.set("GHS - Ghanaian Cedi")  # Change from USD
self.to_combo.set("USD - US Dollar")        # Change from EUR
```

Adjust window size:
```python
# In CurrencyView.__init__()
self.root.geometry("950x720")  # Width x Height
self.root.minsize(600, 550)    # Minimum size
```

Troubleshooting
Common Issues

1. "Failed to fetch currency rates"**
- Check your internet connection
- Verify API is accessible: https://api.exchangerate-api.com/v4/latest/USD
- Check firewall settings

2. "Module 'requests' not found"**
```bash
pip install requests
```

3. Font not displaying properly**
- The app uses Arial (system font)
- On Linux, install: `sudo apt-get install fonts-liberation`

4. Window too small/large**
- Resize manually or adjust in code (see Customization)

 Requirements
Create a `requirements.txt` file:
```
requests>=2.28.0
```

Install all dependencies:
```bash
pip install -r requirements.txt
```
